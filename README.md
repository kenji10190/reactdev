JSとReactで作った○×ゲームです。
ユーザーは先攻でOを使ってゲームをします。

https://kenji10190.github.io/tictactoe-game/ 

このOXゲームには、アルゴリズムが組まれており、ユーザーが手を打つと自動的に最適な位置にXが打たれます。

alpha-beta剪定で最適化されたMinimaxアルゴリズムがゲームに組まれています。  
また、深さペナルティでXの最適な位置の計算精度を上げています。
